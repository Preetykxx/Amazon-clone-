# 🛒 Amazon-clone: Front-end Web Development

## 📌 About This Project
Excited to share my Amazon Clone built from scratch using pure frontend technologies!

This project replicates the core look and feel of Amazon's homepage.

## ✨ Features
- 🧭 Navigation bar with search and cart icon
- 🖼️ Hero banner section
- 📦 Product cards and grid layout
- 🦶 Footer with multiple sections

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
